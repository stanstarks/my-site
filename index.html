Code I find useful

MALLET [Java]
Sampling-based implementations of Latent Dirichlet Allocation (LDA), Pachinko Allocation, and Hierarchical LDA.
Arbylon [Java]
Hierarchical Dirichlet Process (HDP) implementation with a nice companion paper (great for trying to understand HDP for those familiar with LDA)
scikit-learn [Python]
Python machine learning library with quite a few worked examples and decent documentation. The plots will also impress your colleagues at group meetings. At least, if they're used to ROOT plots
PyMC [Python]
The best MCMC estimation I've found in python.
Lomb-Scargle Periodogram [Matlab]
There are tons of LSP implementations floating around the web. After much testing this past summer, I found that this implementation consistently produces the correct LSP in the shortest amount of time for Matlab.
Multi-layer Perception [C]
A great little tutorial on implementing a feed-forward back-prop ANN.
